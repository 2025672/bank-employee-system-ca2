# Bank Employee Management System

CCT College Dublin — HDip in Computing  
Assignment: System Modelling & Build · Package `CA_2`

## Run (NetBeans)

Open as Maven project → Run `CA_2.Main`

## Run (terminal)

```bash
mvn compile exec:java
```

## Features

- Read `Applicants_Form.txt`
- Sort names (recursive merge sort) — first 20 displayed
- Search by name (recursive binary search)
- Add employee with validation
- Employee hierarchy binary tree (level-order)
- Console menu using Java Enums

## Data file

`Applicants_Form.txt` (project root and `src/main/resources/`)
